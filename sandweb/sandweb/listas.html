Linguagens Orientadas a Objetos

  1. Java
    • Linguagem de programação amplamente utilizada para desenvolvimento de aplicações empresariais e móveis (Android). Possui uma vasta biblioteca e é conhecida pela sua portabilidade (escreva uma vez, execute em qualquer lugar).
  2. Python
    • Linguagem de alto nível que é fácil de aprender e usar. Muito popular em ciência de dados, inteligência artificial e automação.
  3. C#
    • Linguagem de programação desenvolvida pela Microsoft, usada principalmente no desenvolvimento de aplicativos para Windows e jogos com o Unity.
  4. PHP
    • Linguagem de script amplamente usada no desenvolvimento de sites dinâmicos e aplicativos web, especialmente com o framework Laravel.

Frameworks para Front-End

React Native
Framework para desenvolvimento de aplicativos móveis utilizando JavaScript e React.
Vue.js
Framework progressivo para construção de interfaces de usuário e aplicativos de página única (SPA).
Bootstrap
Framework front-end para desenvolvimento de sites responsivos e móveis, com base em HTML, CSS e JavaScript.
Angular
Framework para construção de aplicativos web dinâmicos, desenvolvido e mantido pelo Google.
